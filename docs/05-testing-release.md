# 测试与发布

## 测试分层

| 层级 | 工具 | 覆盖内容 |
|---|---|---|
| 单元测试 | JUnit | UseCase、统计逻辑、日期计算。 |
| 数据库测试 | Room test | DAO、唯一约束、迁移。 |
| UI 测试 | Compose UI Test | 今日记录、日历、历史、设置主路径。 |
| 集成测试 | Android Instrumented Test | 加密数据库、DataStore、Biometric 边界。 |
| 手动验证 | 真机 | 通知权限、解锁体验、导出文件。 |

## 必测场景

### 记录

- 当天无记录时创建记录。
- 当天有记录时编辑记录。
- 心情为空时不能保存。
- 正文为空但心情不为空时可以保存。
- 保存失败时输入不丢失。

### 日历

- 当前月无记录。
- 当前月有多条记录。
- 跨月切换。
- 点击日期进入详情。

### 搜索

- 关键词命中正文。
- 关键词无结果。
- 按心情筛选。
- 搜索和筛选组合。

### 统计

- 空数据。
- 单日数据。
- 连续多日数据。
- 中断后重新开始连续记录。
- 跨月连续记录。

### 安全

- 默认启用隐私锁。
- 解锁成功进入主界面。
- 解锁失败保持锁定。
- 数据库密钥读取失败暴露错误。
- 数据库打开失败暴露错误。

### 导出

- JSON 导出成功。
- CSV 导出成功。
- 导出路径不可写时失败。
- 导出内容字段完整。

## 自动化命令建议

后续工程创建后，至少保留这些命令：

```powershell
.\gradlew testDebugUnitTest
.\gradlew connectedDebugAndroidTest
.\gradlew lintDebug
```

后端或 JVM 单元测试运行时限制 60 秒，避免测试卡死。

## 发布检查

- 应用名、图标、包名确认。
- 隐私政策确认。
- 通知权限说明确认。
- 无网络权限，除非后续版本明确需要。
- 日志中无日记正文和密钥。
- 数据库 schema 已导出。
- release 构建可安装。

## 版本规划

### MVP

- 本地日记。
- 心情记录。
- 日历。
- 历史搜索。
- 基础统计。
- 隐私锁。
- 提醒。
- JSON 和 CSV 导出。

### v1.1

- 图片附件。
- 日记模板。
- 自定义心情。
- 自定义活动标签。

### v2

- 可选云同步。
- 端到端加密备份。
- 天气和位置记录。
- 更丰富的数据可视化。
